## v1.2.0 更新说明

- 修复 Alibaba 图片位于 `alicdn.com` 时全部下载失败的问题。
- 补充 Amazon 图片 CDN 权限，并增加图片响应类型与 HTTP 状态诊断。
- 从旧版本升级后，请在 `chrome://extensions` 或 `edge://extensions` 点击一次“重新加载”，并接受新增的网站访问权限。

此扩展用于公开可访问的店铺前台页面，不需要登录店铺后台。Alibaba 若将具体商品详情页重定向至登录页，则需要登录普通买家账号；这由平台控制，扩展不会绕过。

1. 在 Chrome 或 Edge 的扩展管理页开启开发者模式，加载本文件夹。
2. 打开公开的 Alibaba.com 或 Amazon 店铺商品列表页，例如供应商店铺的 `All products` 页面。
3. 点击扩展，填写 ZIP 名称，然后选择“采集当前列表全部商品”。
4. 扩展只会读取 Alibaba 的真实 `www.alibaba.com/product-detail/*.html` 商品链接，或 Amazon 的 ASIN 商品链接；会跟随可识别的下一页，并在每个公开商品详情页提取主图和副图。
5. 浏览器会下载一个 ZIP；每个商品有独立文件夹，名称来自简写商品标题，ZIP 内还包括 `manifest.json`。

如店铺的“下一页”是 JavaScript 按钮而非链接，或商品卡片没有标准产品链接，请把该店铺 URL 发给开发者以添加对应页面规则。
